# 🤖 AI Urban Delivery Robot Simulator

A Python-based intelligent path planning simulator where a robot navigates a city grid to deliver packages using classic AI search algorithms.

## 🚀 Features
- Dynamic random city grid generation
- 5 AI search algorithms implemented from scratch
- Real-time robot animation using Tkinter GUI
- Performance comparison: cost, nodes explored & time

## 🧠 Algorithms
| Algorithm | Strategy | Optimal? |
|-----------|----------|----------|
| BFS | Level by level | ✅ |
| DFS | Depth first | ❌ |
| UCS | Minimum cost | ✅ |
| Greedy | Heuristic only | ❌ |
| A* | g(n) + h(n) | ✅ |
